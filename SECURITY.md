# Security Policy

## Purpose

This repository is a documentation template. It should not contain live secrets, private target inventories, customer data, exploit payloads, or active credentials.

## Reporting a problem with this template

Open an issue or pull request for:

- unclear authorization language;
- unsafe wording that could imply permission where none exists;
- missing data-handling guardrails;
- provider-safety ambiguity;
- placeholders that encourage oversharing;
- template examples that look like real secrets or real targets.

Do not post real secrets, tokens, private keys, cookies, customer data, or sensitive logs in issues.

## Reporting a vulnerability found during an engagement

Do not report real engagement findings in this public template repository unless the finding is already public and fully sanitized.

Use the private engagement record, the authorized reporting channel, and the report templates under `engagement-context/reports/`.

## Public repository warning

A filled copy of this template may disclose sensitive operational information. Keep filled engagement contexts private unless they have been deliberately sanitized for public release.
